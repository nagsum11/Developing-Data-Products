Application Detail

For this assignment I have chosen the number addition application. This application consists of two Text Boxes and a Go button. Application takes two integer input and add them up. You will have to input two integer numbers to Text Boxes. After that hit "Go" button, which will instruct application to do the sum of two integers provided. It displays the addition of two integers as a result.

Application Error Handling

Application shows "NA" if proper input is not provided in two Text Boxes. Example:- If you put number 10 on Text Box 1 and put Spaces or No value in Text Box 2 then output is "NA". "NA" means Not applicable. You cannot add spaces to Integer or vice-versa and that's why the result is "NA".

 shinyUI( pageWithSidebar(
